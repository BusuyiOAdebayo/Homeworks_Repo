Physical Properties Calculation/Estimation/Prediction

Properties are estimated at gas/vapor, liquid, solid and multiphase states

Results are compared using R2 and RMSE values

This is a software application used for calculating, estimating or predicting physical properties of materials (elements and compounds) for the three different three states of matter and their multiphase.

To use this software application, you need to know the state of your material, then select the name of your material, select the property you want to estimate, select the conditions for the estimation and finally the estimation.
